# quail-page

Source for the [quail](https://github.com/spriteCloud/quail) landing
site, published via GitHub Pages at
<https://spritecloud.github.io/quail-page/>.

- `index.html` — landing page
- `docs.html` — command reference + recipes
- `img/`, `src/` — assets used by both pages
- `.github/workflows/pages.yml` — build + deploy on every push to
  `main`

Edits to the HTML / CSS land live within ~30s of a merge to `main`.

The runtime product lives at <https://github.com/spriteCloud/quail>.
